User bisa login sebagai admin maupun sebagai customer.
Jika login sebagai admin maka user harus mengisi halaman login seperti (username, password) namun email yang tertera diside bar atas itu harus email yang digunakan ketika ia login.
Setelah masuk user akan diarahkan ke halaman dashboard admin.

Setelah itu user ke halaman Manajemen produk, halaman tersebut harus bisa responsif seperti ketika button tambah produk di klik maka user bisa menambahkan produk dan ketika button edit di klik maka user bisa mengedit data pada tabel manajemen produk dan ketika button hapus di klik maka produk tersebut akan terhapus, fitur filter dan pencarian harus responsif juga ya.

Setelah itu user ke halaman stok masuk, halaman tersebut harus bisa responsif seperti ketika mengisi input stok masuk maka data yang dimasukan akan masuk ke tabel riwayat stok masuk. (stok yang baru datang/barang baru).

Setelah itu user ke halaman stok keluar, halaman tersebut harus responsif seperti ketika mengisi input stok keluar maka data yang dimasukan akan masuk ke tabel riwayat stok keluar. (stok keluar adalah stok yang sudah terjual).

Setelah itu user ke halaman riwayat dan aktivitas, halaman tersebut harus respponsif, ketika ada aktivitas menambahkan produk maka akan terekam di riwayat dan aktivitas, ketika halaman manajemen produk di edit maka akan terekam di riwayat dan aktivitas, ketika ada barang terjual maka akan terekam di riwayat dan aktivitas, dan ketika ada perubahan harga maka akan terekam juga di riwayat dan aktivitas.

Setelah itu user ke halaman laporan penjualan, halaman tersebut harus responsif, laporan penjualan ini untuk melihat barang apa saja yang terjual dan barang apa saja yang paling laku, user juga bisa melihat total barang terjual, total modal, total pendapatan, total profit, dan user juga bisa melihat laporan penjualan berdsarkan tanggal.
